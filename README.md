# simple-blog-layout-bootstrap-5
#  Simple Blog Layout (Bootstrap 5)

This project is a clean and responsive **blog layout** built using **HTML + Bootstrap 5**.
It includes a navbar, hero section, blog cards, sidebar, and footer — perfect for beginners learning Bootstrap.

---

##  Features

*  Fully responsive layout
*  Clean & modern UI
*  Blog cards section
*  Sidebar with About + Categories
*  Simple footer
*  Quick to customize

---

##  Project Structure

```
/blog-project
│── index.html
```

(You can add `style.css` or `script.js` later if needed.)

---

##  Technologies Used

* HTML5
* Bootstrap 5 (CDN)

---

##  How to Run

1. Download or clone the project
2. Open **index.html** in your browser
3. Or use **VS Code + Live Server** to run it

---

##  Sections Included

* **Navbar:** Simple dark header
* **Hero Section:** Title + description
* **Blog Posts:** Two sample blog cards
* **Sidebar:** About Me + Categories
* **Footer:** Dark theme footer
